PEMAHAMAN DASAR

Halo, ini adalah web yang saya buat untuk mecurahkan serta menambah wawasan dalam belajar web progamming. Yap, pasti banyak orang yang tidak asing dengan satu bahasa pemograman yang menjadi satu pondasi umum seorang progamming,

Perlu diketahui HTML bukan salah satu bahasa pemrogaman melainkan menjadi satu pondasi penting dalam sebuah web.

Disini saya akan membagikan beberapa materi yang sudah saya pelajari, apabila ada kesalahan atau kurang tepatnya mengenai aturan penulisan ataupun pemaknaan saya selaku penulis meminta saran dan kritik pembaca.

HYPER TEXT MARK LANGUAGE, merupakan bahasa markup standar yang digunakan untuk membuat dan menyusun halaman halaman web.

Meskipun sering disangka bahasa pemograman, HTML sebenarnya bukanlah bahasa yang bisa menjalankan logika atau fungsi dinamis , melainkan untuk mengatur struktur dan konten sebuah halaman

Fungi Utama HTML, membuat struktur halaman web, menandai bagian bagian konten, menyisipkan multimedia, membuat hyperlink.

DEFINISI HTML

HTML => PONDASI WEB

head

berisi domain / alamat dari suatu website atau bisa juga dijadikan suatu penghubung antara kerangka bahasa html atau bahasa pemrogaman lainnya.

title

untuk memberi nama / judul suatu website

body

berisikan isi / bagian tubuh dari suatu bagian website, yang dimana pada *body ini nantinya akan menjadi suatu kerangka penting dalam membuat website.

footer

bagian kaki dari suatu kerangka website berisikan info dari suatu website / si penulis.

Element HTML

Pada umumnya HTML terdiri dari element yang dibagi menjadi :

  • TAG

    adalah untuk memberi tanda pada bagian web

  • ATRIBUT

    untuk memberi memberi informasi tambahan yang ditulis pada tag pembuka

  • VALUE

    nilai dari suatu atribut

Berikut Beberapa Tag pada HTML dasar :

  1. memberi efek bold < b >

  2. memberi efek italic < i >

  3. teks bergaris bawah < u >

  4. teks berukuran lebih kecil < small >

  5. penekanan pada teks(semantik) < em >

  6. teks yang ditandai < mark >

  7. teks yang dihapus < del >

  8. teks yang disisipkan / dipulihkan < ins >

  9. teks yang mengecil kebawah H2 < sub >

  10. teks yang mengecil keatas O2 < sup >

  11. teks variabel yang menandai suatu rumus : v = s / t < var >

Media HTML

HTML memiliki tag yang bisa kita gunakan untuk menyisipkan konten seperti : gambar, audio, video dan bahkan file eksternal seperti PDF atau Youtube.

Gambar tersebut bisa kita tampilan ke dalam sebuah website dengan bantuan sebuah tag |< img>|.

Gambar yang ingin kita tampilkan bisa kita ambil dari sebuah web lain(online) ataupun dapat juga gambar yang kita simpan di directory(offline). Dan perlu dicermati jika kita mengambil suatu media dari online, sebaiknya kita teliti perihal hak cipta dan copyrightnya.
